# Network Ops AI Agent

A local LLM-powered chatbot specialized in network operations, built with Python and Ollama.

## Features
- Conversation history maintained across turns
- System prompt for domain specialization
- 100% local inference (no API costs)

## Setup
1. Install Ollama: `brew install ollama`
2. Pull model: `ollama pull llama3.2`
3. Start Ollama: `ollama serve`
4. Run: `python3 main.py`

## Week 1 Learning Goals
- Python classes and objects
- Subprocess management
- Conversation state management
- LLM prompt engineering basics
